# HVAC Chilled Water Toolkit

A lightweight, browser-based toolkit for HVAC engineers working with chilled water and air systems. Designed for **fast, practical use on site**, not as a replacement for detailed design software.

---

## 🚀 Overview

This app provides a collection of commonly used HVAC and basic electrical calculations in one place, accessible from any device via a web browser.

No installation required. No logins. No nonsense.

---

## ⚙️ Features

### 🧊 Cooling & Water Side

* Cooling capacity (kW) from flow and ΔT
* Flow rate from kW and ΔT
* Chiller COP calculator
* Glycol correction (approximate)
* Metering station / Kvs flow calculation

### 🔁 Pipe & Pump

* Pipe sizing from flow and velocity
* Pump power estimation
* Pump/system curve estimator

### 💨 Air Side / Duct

* Duct sizing (area + round equivalent)
* Duct velocity checker
* Air changes per hour (ACH)
* Airside heating/cooling duty
* Airflow required from duty
* Room cooling load (Quick & Advanced modes)

### ⚡ Electrical

* Current calculation (single & three phase)
* Cable sizing helper (basic estimator)

### 🔄 Converters

* Length
* Weight
* Power
* Speed
* Volume
* Flow rate
* Pressure
* Refrigerant pressure/temperature reference

### 🧮 Engineering Calculator

* Quick general-purpose calculator
* Supports brackets, powers, roots, logs
* Includes percentage function

---

## 📱 Usage

* Open the app in any browser
* Navigate using the tabs
* Enter values — most tools calculate automatically
* Results update instantly

Designed to be:

* Fast
* Simple
* Usable on site with minimal input

---

## ⚠️ Disclaimer

This tool is intended for **quick field calculations and estimates only**.

It does **not replace**:

* Manufacturer data
* Detailed engineering design
* BS 7671 electrical standards
* Full HVAC design calculations

Always verify results before final decisions or sign-off.
