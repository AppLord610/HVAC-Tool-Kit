Cable Sizing Notes

What this checks

Design current, rough current carrying capacity, and voltage drop. It is a quick filter, not a certified design.

What still needs checking

Installation method, cable type, ambient temperature, grouping, thermal insulation, protective device, disconnection time, Zs, short-circuit withstand, volt drop, harmonics, motor starting current, and manufacturer data.

Rule of thumb

If this app suggests a size, treat it as a starting point. Then check the actual tables properly, because electricity does not care about confidence.

Field Guide / How to Use This App Properly

Cooling Calculations

kW = Flow × 4.187 × ΔT. Always use stable readings. Bad sensors = bad answers.

Flow Calculations

Reverse calcs assume accurate ΔT. Low ΔT = high flow = usually a system issue.

Metering Stations / Orifice Plates

Flow = Kvs × √Δp. Use correct Kvs for the exact valve insert. Check DP range — too low = inaccurate, too high = outside design.

Pipe Sizing

Velocity is only a starting point. Always check pressure drop and noise limits.

Pump Calculations

Pump power is estimated. Always verify against pump curves and duty points.

Pump Curve Tool

This plots an estimated curve using H ∝ Q². Good for site checks, not for design sign-off.

Glycol

Reduces performance and increases pressure drop. Always use manufacturer data for final design.

Converters

Useful for site work but always sanity check units — especially flow and pressure.

Refrigerant Slider

Approximate saturation temps only. Real systems need proper pressure-temperature charts.

Cable Sizing

This is a rough estimator. Final sizing must follow BS 7671, including volt drop, Zs, protection and installation method.

Golden Rule

This app helps you think faster. It does not replace proper engineering checks. If it looks wrong, it probably is.
